📖 Hướng dẫn Cài đặt & Sử dụng

1️⃣ Cài đặt Server

npm install
npm start

2️⃣ Chọn LLM Provider

  • Ollama (Khuyên dùng - Hoàn toàn miễn phí)
    • Download: https://ollama.ai
    • Chạy: ollama run mistral
  • Groq API (Free Tier)
    • Đăng ký: https://console.groq.com
    • Cập nhật .env
  • HuggingFace (Free Tier)
  • Cohere (Free Tier)

3️⃣ Cấu hình Zalo OA

  1. Truy cập https://oa.zalo.me
  2. Vào Settings → API → Webhook
  3. Điền Webhook URL: http://your-server:3000/webhook?token=YOUR_TOKEN
  4. Lấy Access Token và điền vào .env

4️⃣ API Endpoints

  • POST /api/documents/upload-form - Upload tài liệu
  • GET /api/documents - Lấy danh sách tài liệu
  • DELETE /api/documents/:id - Xóa tài liệu
  • POST /api/test-llm - Test LLM

🔐 Bảo mật

  • Sử dụng HTTPS khi deploy production
  • Giữ bí mật các API keys trong .env
  • Xác thực webhook token từ Zalo
